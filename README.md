# Symptom Checker Chatbot

A multilingual AI-powered symptom checker for headache prediction.

## Features

* Headache type prediction
* Remedies and precautions
* English, Hindi and Marathi support
* Speech-to-text input
* Text-to-speech output
* Dark mode
* Flask backend
* Machine Learning model using Random Forest

## Technologies

* Python
* Flask
* Scikit-learn
* Whisper
* HTML
* CSS
* JavaScript

## Run Project

Install requirements:

pip install -r requirements.txt

Run backend:

python app.py

Open frontend/index.html
